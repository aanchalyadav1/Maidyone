Maidyone

Maidyone is a full-stack web application for managing service-based operations like users, workers, and bookings through an admin dashboard.

---

Tech Stack

- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB

---

│
├── backend/
│   ├── src/

│   └── .env

---

Setup & Run

Clone the repository

git clone https://github.com/aanchalyadav1/Maidyone.git
cd Maidyone

Run Frontend

cd frontend
npm install
npm run dev

Run Backend

cd backend
npm install
npm run dev
